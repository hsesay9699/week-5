# Online Shopping Node.js Application

This is a Node.js web application built following the **"Node.js: The Complete Guide"** video series (Chapters 4-9). It demonstrates the usage of Express.js, MVC architecture, and basic file-based data storage.

## Features
* **MVC Architecture:** Organized into Models, Views, and Controllers.
* **Admin Features:** Add products with title, image, price, and description.
* **Shop Features:** View product list and add items to a shopping cart.
* **Data Storage:** Uses JSON files (`data/products.json` and `data/cart.json`) to persist data.

## Installation & Usage

1.  **Install Dependencies:**
    Open your terminal in the project folder and run:
    ```bash
    npm install
    ```

2.  **Start the Server:**
    Run the following command:
    ```bash
    node app.js
    ```

3.  **View the App:**
    Open your browser and navigate to:
    `http://localhost:3000`
